# WorldSeed Ideas (Non-Normative)

This directory collects research questions that become
well-defined only after WorldSeed-style world declarations
are made explicit.

The contents of this directory are:

- Non-normative
- Non-binding
- Non-exhaustive
- Not part of the WorldSeed specification

No proposal in this directory constitutes a claim,
a recommendation, or a requirement.

These ideas exist to document
problem spaces that were previously discussed informally,
but which become mathematically or formally approachable
once world assumptions are explicitly declared.

Any future formalization, proof, or theory built upon these ideas
must still independently satisfy the WorldSeed specification
and its axioms.

# WorldSeed Ideas（非规范）

本目录收集的是：  
在 WorldSeed 风格的世界声明前提下，
首次变得“可被严肃形式化”的研究问题。

本目录内容具有以下属性：

- 非规范（non-normative）
- 非承诺（non-binding）
- 非路线图（non-roadmap）
- 不构成 WorldSeed 规范的一部分

此处提出的问题并不代表结论、建议或要求，
仅用于记录那些在世界假设显式化之后，
才具备数学化或形式化可能性的研究空间。

任何基于这些想法展开的理论或证明，
仍需独立满足 WorldSeed 的规范与公理要求。
